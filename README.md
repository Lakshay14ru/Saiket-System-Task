# To-Do List Application

A simple and user-friendly command-line based To-Do List Application developed using Python. This project allows users to efficiently manage their daily tasks by adding new tasks, viewing existing tasks, and marking tasks as completed.

The application is designed for beginners to understand core Python programming concepts such as lists, dictionaries, loops, conditional statements, exception handling, and menu-driven programs.

---

# Features

* Add new tasks dynamically
* View all added tasks
* Mark tasks as completed
* Displays task completion status
* User-friendly menu-driven interface
* Input validation using exception handling
* Stores tasks using dictionaries
* Continuous execution until user exits

# Functionalities

## 1. Add Task

Users can add a new task to the to-do list.

## 2. View Tasks

Displays all tasks along with their current status:

## 4. Exit Application
